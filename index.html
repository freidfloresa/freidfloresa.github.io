Aplikasi Cetak Ijazah dan Transkrip Nilai Akademik

Aplikasi Cetak Ijazah dan Transkrip Nilai Akademik Berbasis Web. Sistem ini dibangun dengan menggunakan bahasa pemrograman PHP untuk server side nya dan juga MySqli untuk database nya, sedangkan untuk front end nya saya menggunakan framework CSS Bootstrap 3.0. Aplikasi ini dibuat untuk mempermudah proses pencetakan ijazah dan transkrip nilai akademik di Unika Weetebula karena dilengkapi dengan fitur import data sehingga menjadi lebih cepat dan efisien.

Sistem Informasi Manajemen Aset

Sistem Informasi Manajemen Aset (SIMAS) berbasis web. Sistem ini dibangun dengan menggunakan bahasa pemrograman PHP dan AJAX untuk server side nya dan juga MySqli untuk database nya, sedangkan untuk front end nya saya menggunakan framework CSS Bootstrap 3.0. Aplikasi ini dibuat untuk mempermudah biro layanan umum dalam mengelola seluruh aset institusi yang ada.
